## Purpose
This org contains repos used by the **GitHub Deveoloper Relations** team for demos at conferences such as [**GitHub Universe**](https://githubuniverse.com/), [**Microsoft Build**](https://build.microsoft.com/) and [**Microsoft Ignite**](https://ignite.microsoft.com/) as well as in numerous customer **Executive Briefings**. This org is also used to store shared assets, such as **GitHub Copilot Custom Agent** definition files that are used across multiple demo environments.
